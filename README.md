# 🎮 Rock Paper Scissors (C++)

A simple console-based Rock Paper Scissors game built using C++.

## Features
- User vs Computer
- Random Computer Choice
- Win/Lose/Draw Logic
- Replay Option

## Concepts Used
- Conditional Statements
- Loops
- Random Number Generation
- Basic Input Validation

## How to Run

Compile:
g++ main.cpp -o rps

Run:
./rps
